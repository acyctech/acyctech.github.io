2017.1.2:习惯养成很重要

用hexo生成博客文章的流程

生成新文章
hexo new “article title”

编辑生成的文章，文章所在路径为：/source/_posts

生成静态页面
hexo generate

本地预览
hexo server
http://localhost:4000/

发布到Github
hexo deploy

注：配置工作已经提前做好了
ps:一个习惯的养成不是一件容易的事，每天一篇文章，也不是很好坚持的，但不管怎样，哪怕只写一个字，也要坚持。先从形式上做到，再深化内容！
